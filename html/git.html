Git

  • git init - инициализировать репозиторий в выбранной директории.
  • git clone 'ссылка на репозиторий' - клонировать репозиторий.
  • git branch 'название ветки' - создать новую ветку.
  • git branch -d 'название ветки' - удалить указанную ветку.
  • git checkout 'название ветки' - переключиться на указанную ветку.
  • git add - добавить созданный, удаленный, измененный файл в репозиторий.
  • git commit -m 'название коммита' - закоммитить изменения на ветке и дать название коммиту.
